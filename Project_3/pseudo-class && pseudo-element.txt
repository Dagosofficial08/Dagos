Daftar Pseudo-class di CSS

1. User Action / Link States

    :hover â†’ saat mouse berada di atas elemen.

    :active â†’ saat elemen sedang diklik.

    :focus â†’ saat elemen mendapat fokus.

    :focus-visible â†’ fokus yang terlihat (misalnya keyboard navigation).

    :focus-within â†’ parent yang punya child sedang fokus.

    :visited â†’ link yang sudah dikunjungi.

    :link â†’ link yang belum dikunjungi.

2. Structural (Posisi dalam DOM)

    :first-child â†’ anak pertama.

    :last-child â†’ anak terakhir.

    :nth-child(n) â†’ anak ke-n.

    :nth-last-child(n) â†’ anak ke-n dari belakang.

    :only-child â†’ satu-satunya anak.

    :first-of-type â†’ elemen pertama dari jenis tertentu.

    :last-of-type â†’ elemen terakhir dari jenis tertentu.

    :nth-of-type(n) â†’ elemen ke-n dari jenis tertentu.

    :nth-last-of-type(n) â†’ elemen ke-n dari belakang dari jenis tertentu.

    :only-of-type â†’ satu-satunya elemen dari jenis tertentu.

    :empty â†’ elemen tanpa konten.

    :root â†’ elemen root dokumen (biasanya <html>).

    :scope â†’ elemen yang jadi scope selector.

3. Form & Input States

    :enabled â†’ input aktif.

    :disabled â†’ input tidak aktif.

    :checked â†’ checkbox/radio dicentang.

    :indeterminate â†’ checkbox dalam state tidak pasti.

    :valid â†’ input valid.

    :invalid â†’ input tidak valid.

    :required â†’ input wajib.

    :optional â†’ input opsional.

    :in-range â†’ nilai input dalam rentang valid.

    :out-of-range â†’ nilai input di luar rentang.

    :read-only â†’ input hanya baca.

    :read-write â†’ input bisa diubah.

    :placeholder-shown â†’ input sedang menampilkan placeholder.

4. Language & Direction

    :lang(xx) â†’ elemen dengan atribut lang tertentu.

    :dir(ltr|rtl) â†’ arah teks tertentu.

5. Logical / Negation

    :not(selector) â†’ elemen yang tidak cocok dengan selector.

    :is(selector) â†’ elemen yang cocok dengan salah satu selector.

    :where(selector) â†’ mirip :is tapi tanpa menambah specificity.

    :has(selector) â†’ elemen yang punya child/descendant tertentu.

6. Miscellaneous

    :target â†’ elemen yang jadi target URL fragment (#id).

    :fullscreen â†’ elemen dalam mode fullscreen.

    :current, :past, :future â†’ pseudo-class untuk media (misalnya cue di video/audio).
